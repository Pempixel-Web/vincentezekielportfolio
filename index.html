Honeyland School Website

I designed and developed a modern and responsive website for Honey Land School to strengthen their online presence and make information easily accessible to parents and students. The website features a clean layout with sections for school information, admissions, academics, news updates, and contact details. It was built with a user-friendly interface, fast loading speed, and mobile responsiveness to ensure a smooth experience across all devices.

AAJ Express Logistics Website

I designed and developed a modern logistics website inspired by the structure and services of AAJ Express, a global shipping and delivery company. The platform was built to present logistics services clearly while providing a smooth user experience for tracking shipments, exploring delivery options, and requesting shipping services. The website includes structured service sections such as international shipping, local and interstate delivery, and cargo logistics, along with responsive design and optimized performance for both desktop and mobile users. The project focuses on clear navigation, service presentation, and trust-building elements that logistics companies need to communicate reliability and efficiency in transporting goods locally and internationally.

PixelPay – Digital Banking Dashboard UI

PixelPay is a modern digital banking dashboard designed to simulate the interface of a real fintech application. The project features a user account overview, transaction history table, balance display with Nigerian currency formatting, and quick financial actions such as airtime purchase, data subscription, electricity payment, and bill settlement. The interface focuses on clean fintech design patterns, responsive layout, and interactive UI elements to create a smooth user experience similar to real Nigerian banking applications.


Tatu Property – Real Estate Investment & Property Showcase Website

Tatu Property is a modern real estate website developed to present large-scale property developments, investment opportunities, and residential projects through a clean and structured digital platform. The website features sections for property listings, project details, image galleries, and company news to keep investors and potential buyers informed about ongoing developments. The interface emphasizes strong visual presentation, intuitive navigation, and responsive design to ensure a smooth browsing experience across devices. This project demonstrates practical skills in front-end development, layout structuring, and building scalable websites for real estate companies and property developers.
